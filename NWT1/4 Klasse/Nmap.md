# Nmap

Nmap installieren:

sudo apt install nmap (auf Linux)



Mit nmap -sn <subnetzmaske> sieht man alle Leute in seinem eigenen Subnetz die man pingen kann

-sn ... subnetzmask

#### Um eine Ip Addresse zu scannen:

nmap <ip addresse>

```bash
nmap <domain>
```
